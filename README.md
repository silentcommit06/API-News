# API News

A simple React app for searching live news articles with the NewsData API.

## Setup

Create a `.env` file in the project root:

```env
REACT_APP_NEWS_API_KEY=your_api_key_here
```

## Run

From `C:\Users\mahta\Documents\API-News`, start the app with:

```powershell
cmd /c npm start
```

Then open [http://localhost:3000](http://localhost:3000).
